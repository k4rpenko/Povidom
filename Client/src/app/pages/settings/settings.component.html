Povidom
Povidom — a modern social network based on Angular and ASP.NET (C#) Povidom is my pet project, a social network created for exchanging opinions, news, and interaction between users in a convenient and intuitive environment. The project was developed using a modern technology stack: the frontend is built on Angular, and the backend is implemented on ASP.NET using the C# programming language.
